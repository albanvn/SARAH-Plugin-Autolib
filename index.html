Le plugin 'Autolib' permet de connaitre les voitures/places disponibles à vos stations favorites



Configuration et paramétrages du plugin.

Il faut rentrer dans la configuration du plugin les numéros de station autolib qui vous interessent.
Pour cela, consulter le fichier autolib_stationlist.txt fournie dans le répertoire du plugin.
Il contient la liste des adresses des stations autolib et leurs numéros d'identification.
Vous pouvez ajouter autant de numéros de stations que vous voulez dans le champs "stations" de la configuration du plugin, chaque station est séparé par une virgule.


Utilisation.



Commandes reconnues par SARAH.

  • Places autolib disponibles:SARAH recherche les places autolib disponibles
  • Voitures autolib disponibles:SARAH recherche les voitures autolib disponibles
  • Est-ce qu'il y a des places autolib disponibles:SARAH recherche les places autolib disponibles
  • Est-ce qu'il y a des voitures autolib disponibles:SARAH recherche les voitures autolib disponibles
